NeoVimRC
========

How to use
----------

1. Clone this repo (with `Vundle` submodule by `--recursive` flag):

  ```bash
  $ git clone --recursive https://github.com/unclechu/neovimrc ~/.config/nvim
  ```

2. Install [Vundle](https://github.com/gmarik/Vundle.vim) dependencies:

  ```bash
  $ nvim +BundleInstall
  ```

3. Enjoy!
